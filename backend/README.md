# CSAO Recommendation Engine

This repository powers the Machine Learning API backend for the CSAO Rail Recommendation System.
It is built on FastAPI and utilizes LightGBM for real-time contextual scoring.

For the main project files and Frontend React application, please refer to the primary GitHub Repository.
